# Enterprise Multimodal RAG System

Production-grade Retrieval-Augmented Generation system
for enterprise knowledge bases.

## Overview
This system allows querying enterprise documents such as
company policies, SOPs, and procedural manuals that contain
both textual content and embedded workflow diagrams.

## Key Features
- Text + Image PDF ingestion
- Multimodal FAISS vector index
- Hybrid retrieval + reranking
- Source-grounded answers
- Diagram retrieval from PDFs

## Use Cases
- Company policy Q&A
- SOP clarification
- Workflow understanding
- Compliance documentation search

## Tech Stack
LangChain, FAISS, HuggingFace, PyMuPDF,
FastAPI, Streamlit
